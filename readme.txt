# Using convolutional neural network to classify bean images. 

## Please find the following instructions to run the demo. 

1. Upload the folder (Data) and the code (Beans Classification CNN.ipynb) to your google drive and put them under the same folder.
2. Go to https://colab.research.google.com/ and open the code. 
3. Click the Play icon in the left gutter of the cell to run the cell. Find more Colab features here: https://colab.research.google.com/notebooks/basic_features_overview.ipynb#scrollT
